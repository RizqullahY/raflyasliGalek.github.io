Portfolio

Project Terbaru

Berikut beberapa gambar dari proyek-proyek yang telah saya kerjakan beberapa waktu terakhir. Mencakup beberapa aspek, mulai dari pengembangan Frontend dan Backend dari web maupun mobile, hingga desain API dan manajemen database.

Manajemen Reservasi - Admin | Reservasi Bengkel

Aplikasi yang saya buat menggunakan React untuk frontend dan .NET untuk backend API-nya. Dengan tema reservasi bengkel, aplikasi ini memungkinkan admin untuk mengelola seluruh data, sementara pelanggan dapat melakukan reservasi kendaraan, mengisi data diri, serta menyertakan detail kendaraan mereka.

Layanan Bengkel Central Otomotif

Aplikasi yang saya buat ini menggunakan React untuk frontend dan Node.js untuk backend API-nya. Mirip dengan aplikasi sebelumnya, aplikasi ini memiliki fitur yang lebih sederhana, hanya mencakup tampilan layanan yang tersedia serta fitur login dan registrasi untuk pengguna baru. Fitur-fitur ini dirancang untuk dapat diintegrasikan dengan aplikasi sebelumnya.

Transaksi Reservasi Bengkel

Aplikasi yang saya buat menggunakan framework PHP Laravel dengan tema yang sama, yaitu reservasi bengkel. Namun, aplikasi ini dirancang khusus untuk pengguna dengan peran kasir. Di dalam aplikasi ini, terdapat fitur untuk mengelola kumpulan data reservasi pelanggan serta transaksi, memudahkan kasir dalam melakukan tugas-tugas administratif terkait reservasi dan pembayaran.

Layanan Bengkel Central Otomotif - Mobile

Aplikasi yang saya buat menggunakan React Native untuk frontend dan .NET untuk backend API-nya. Aplikasi ini merupakan versi mobile dari layanan Bengkel Central Otomotif. Berbeda dari aplikasi sebelumnya, aplikasi ini tidak memiliki fitur login atau registrasi. Aplikasi ini menampilkan bagian home, deskripsi singkat tentang bengkel serta daftar layanan yang tersedia.
